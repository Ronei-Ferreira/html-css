Usando Links Internos

Os Links internos vai fazer a ligação e a relação da nossa pagina com as outras paginas que estão dentro do nosso servidor que condiz e diz respeito a nossa propria pagina

Para criar link interno, temos que ter outras paginas (para servir de links para navegação

Neste caso foi criado a pag022.index, para servir de link para acesso atravé da minha primeira pagina

Dentro da primeira pagina, criamos um paragrafo onde vamos colocar a descrição com o link da pagina002, criamos uma ancora usando a tag 'a' e dentro de ref colocamos o arquivo (link) da proxima pagina, como no exeplo abaixo

Esta é a minha primeira pagina.
se você quiser, pode também acessar a minha segunda página

Quando usamos links intermos, não precisamos colocar a URL inteira do link, apenas o nome da pasta, porém as os arquivos das paginas/links ter que esta dentro da mesma pasta da primeira página

Para descrever que a proxima pagina sera a sequencia de navegação, colocar a paravra 'next dentro de rel dentro do arquivo da proxima pagina

Para acessar uma terceira pagina pag003, criar uma pasta dentro do arquivo e criar um código para pag003. dentro do código da primeira pagina, criar a ancora tag 'a' para pagina 003, não vai aparecer pois ela esta dntro de uma subpasta

Esta é a minha terceira página

Para poder navegar sem pelas paginas do site sem precisar ficar abrindo novas abas, colocar a rel='next' e target='self'. Desta forma você ira navegar nas paginas do site sem ficar abrindo novas abas

SE NÃO COLOCAR O TARGET=SELF, A PAGINA ABRIRA EM UMA ABA NOVA E NÃO NA MESMA PAGINA
