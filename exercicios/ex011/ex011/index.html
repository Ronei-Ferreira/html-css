Imagens Dinamicas

Tente abrir esse site em várias dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador

Devemos inserir imagens em tamahos dferentes para que quando aberta em dispositivos diferentes (celular, computador, tablet, tvs, etc.., as imagens se ajustem ao tmano da tela de cada dispositivo, e o usuario final não tenha que ficar utilizando a barra de rolagem para conseguir visualizar a imagem toda, pois isso faz com que os usuarios não utilizeo o seu site

Para colocar as imagens em varios tamanos, devemosutilizar a tad 'picture, e dentro desta tag, inserir sources na sequencia de tamanho que as imagens seram utilizada conforme a imagem abaixo do códico. para ajustar a imagem grande em que não seja necessario aparecer a barra de rolagem, sempre acrescentar 0,50 px no valor dentro de max-width que este é o valir de espaço da barra de rolagem.
