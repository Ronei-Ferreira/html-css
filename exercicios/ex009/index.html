Listas Ordenadas

Existe uma TAG ol que Ordema as lintas. Para cada item de Lista existe a TAG li que ordena a lista.

Caso esqueça de colocar um item na ordem correta, é só colocar a TAG 'li' na posição que deseja, que a lista se organiza sozinha

A TAG de fechamento da 'li' se tornou NÃO OBRIGATÓRIA
